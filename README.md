# koydo-toeic-speaking-writing

TOEIC Speaking & Writing exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM008 |
| Name | TOEIC Speaking & Writing |
| Country | Global |
| Category | Language proficiency |
| Tier | Tier 2 |
| Wave | Wave 3 |
